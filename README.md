本项目是SoLive（珊瑚动听）直播PC客户端（观众端），基于以下组件：

1. mediasoupclient
2. WebRTC (M120/6099)
3. Qt 5.14.2
4. Windows SDK 10.0.22621.0
5. socket-io-client
6. nlohmann-json 3.11.2
7. FFmpeg
8. openssl
9. boost
